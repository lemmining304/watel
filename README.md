# Watel

## Conheça o Watel

Watel é uma bola que desvia dos erros 404

## Como instalar

Para fazer isso, segue os passos:

1. Vá no Termux e digite:

```
pkg install git nodejs

```

2. Instale com:

```
git clone https://github.com/lemmining304/watel.git

```

3. Vá para a pasta:

```
cd watel

```

4. E rode:

```
node server.js

```

Você vai ver:

```
Servidor rodando em localhost:3000

```

Copie o link, e coloque no navegador, vai aparecer o Jogo

E lembre-se:

> Desvie dos erros 404
